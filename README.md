# W3D1 Assignment: The Secure Bank Account

### Your Mission
Open `src/BankAccount.java` and build a secure, encapsulated class.
1. All attributes must be `private`.
2. You must build a Constructor so `Main.java` can create the object.
3. Your `deposit` and `withdraw` methods must contain `if/else` logic to stop hackers from depositing negative money or withdrawing more than they have!

*Warning: If you try to run this code right now, it will fail because `Main.java` is looking for a constructor that doesn't exist yet!*
